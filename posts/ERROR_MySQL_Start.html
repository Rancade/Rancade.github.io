🎯 疑难解答 ​

Q: 端口被占用怎么办？

bash
sudo netstat -tulnp | grep 3306 \
| awk '{print $7}' | cut -d'/' -f1 | xargs kill -9
1
2

Q: 忘记密码了？

bash
sudo mysqld_safe --skip-grant-tables &
mysql -u root
# 然后在MySQL中执行：
UPDATE mysql.user SET authentication_string=null WHERE User='root';
FLUSH PRIVILEGES;
exit;
